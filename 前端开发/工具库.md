# 工具&软件&网站

## curl

- 和服务器交互的工具
- 支持多种协议
- curl https://www.sogou.com/ 显示网站响应值
- curl -I https://www.sogou.com/ 获取 http header

## whistle

- 跨平台网络调试工具
  - 需要 SwitchyOmega 插件
  - nodejs 开发
  - 支持抓包、重放、替换、修改等
